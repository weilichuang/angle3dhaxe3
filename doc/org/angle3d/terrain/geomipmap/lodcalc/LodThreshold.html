interface org.angle3d.terrain.geomipmap.lodcalc.LodThreshold

Available on all platforms

Implementors
SimpleLodThreshold
SimpleLodThreshold

Just multiplies the terrain patch size by 2. So every two patches away the camera is, the LOD changes.

Calculates the LOD value based on where the camera is. This is plugged into the Terrain system and any terrain using LOD will use this to determine when a patch of the terrain should switch Levels of Detail.
