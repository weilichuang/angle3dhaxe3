class org.angle3d.terrain.heightmap.MidpointDisplacementHeightMap extends AbstractHeightMap

Available on all platforms

MidpointDisplacementHeightMap generates an heightmap based on the midpoint displacement algorithm. See Constructor javadoc for more info. @author cghislai

Instance Fields

function new(size:Int, range:Float, persistence:Float, seed:Int):Void

The constructor generates the heightmap. After the first 4 corners are randomly given an height, the center will be heighted to the average of the 4 corners to which a random value is added. Then other passes fill the heightmap by the same principle. The random value is generated between the values -range and range. The range parameter is multiplied by the persistence parameter each pass to smoothen close cell heights. Extends this class and override the getOffset function for more control of the randomness (you can use the coordinates and/or the computed average height to influence the random amount added.

@throw JMException if size is not a power of two plus one.

size

The size of the heightmap, must be 2^N+1

range

The range in which randomness will be added. A value of 1 will allow -1 to 1 value changes.

persistence

The factor by which the range will evolve at each iteration. A value of 0.5f will halve the range at each iteration and is typically a good choice

seed

A seed to feed the random number generator.
