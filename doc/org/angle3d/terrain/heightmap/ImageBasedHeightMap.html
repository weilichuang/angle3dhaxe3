class org.angle3d.terrain.heightmap.ImageBasedHeightMap extends AbstractHeightMap implements ImageHeightmap

Available on all platforms

ImageBasedHeightMap is a height map created from the grayscale conversion of an image. The image used currently must have an equal height and width, although future work could scale an incoming image to a specific height and width.

@author Mike Kienenberger @version $id$

Instance Fields

function new(colorImage:BitmapData, ?heightScale:Float, ?flipX:Bool, ?flipY:Bool):Void

Creates a HeightMap from an Image. The image will be converted to grayscale, and the grayscale values will be used to generate the height map. White is highest point while black is lowest point.

Currently, the Image used must be square (width == height), but future work could rescale the image.

colorImage

Image to map to the height map.
