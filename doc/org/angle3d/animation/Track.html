interface Track

package org.angle3d.animation

implemented by BoneTrack, SpatialTrack

Methods

clone ():Track

This method creates a clone of the current object.

Returns:

a clone of the current object

getKeyFrameTimes ():Vector<Float>

Get the times in seconds for all keyframes.

All keyframe times should be between 0.0 and {#getLength() length}. Modifying the provided array is not allowed, as it may corrupt internal state.

Returns:

the keyframe times

getLength ():Float

Returns:

the time of the track

setTime (time:Float, weight:Float, control:AnimControl, channel:AnimChannel):Void

Sets the time of the animation.

Internally, the track will retrieve objects from the control and modify them according to the properties of the channel and the given parameters.

Parameters:

time

The time in the animation

weight

The weight from 0 to 1 on how much to apply the track

control

The control which the track should effect

channel

The channel which the track should effect
