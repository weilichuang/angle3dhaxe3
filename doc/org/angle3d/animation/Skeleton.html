class Skeleton

package org.angle3d.animation

Skeleton is a convenience class for managing a bone hierarchy. Skeleton updates the world transforms to reflect the current local animated matrixes. A Skeleton can only one rootBone

Constructor

new (?boneList:Vector<Bone>)

Creates a skeleton from a bone list. The root bones are found automatically.

Note that using this constructor will cause the bones in the list to have their bind pose recomputed based on their local transforms.

boneList

The list of bones to manage by this Skeleton
