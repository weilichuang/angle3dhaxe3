Collidable

Interface for Collidable objects. Classes that implement this interface are marked as collidable, meaning they support collision detection between other objects that are also collidable.

CollisionData

CollisionData is an interface that can be used to do triangle-accurate collision between bounding volumes and rays.

CollisionResult

A CollisionResult represents a single collision instance between two {Collidable}. A collision check can result in many collision instances (places where collision has occured).

CollisionResults

CollisionResults is a collection returned as a result of a collision detection operation done by Collidable.
