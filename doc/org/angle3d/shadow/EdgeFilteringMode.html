abstract org.angle3d.shadow.EdgeFilteringMode(Int)

Available on all platforms

ShadowEdgeFiltering specifies how shadows are filtered

Instance Fields

var Bilinear:EdgeFilteringMode

Bilinear filtering is used.

var Nearest:EdgeFilteringMode

Shadows are not filtered. Nearest sample is used, causing in blocky shadows.

var PCF:EdgeFilteringMode

3x3 percentage-closer filtering is used. Shadows will be smoother at the cost of performance

function toInt():Int
