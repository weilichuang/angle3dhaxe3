package org.angle3d.effect.cpu.influencers

DefaultParticleInfluencer

This emitter influences the particles so that they move all in the same direction. The direction may vary a little if the velocity variation is non zero. This influencer is default for the particle emitter.

EmptyParticleInfluencer

This influencer does not influence particle at all. It makes particles not to move.

IParticleInfluencer

An interface that defines the methods to affect initial velocity of the particles. 用于影响粒子初次的速度

NewtonianParticleInfluencer

This influencer calculates initial velocity with the use of the emitter's shape.
