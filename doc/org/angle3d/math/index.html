Line

Line defines a line. Where a line is defined as infinite along two points. The two points of the line are defined as the origin and direction.

LineSegment

LineSegment represents a segment in the space. This is a portion of a Line that has a limited start and end points.

Matrix3f

3X3矩阵，主要用于旋转操作

Matrix4f

Matrix4f defines and maintains a 4x4 matrix in row major order. This matrix is intended for use in a translation and rotational capacity. It provides convenience methods for creating the matrix from a multitude of sources.

Plane

Plane defines a plane where Normal dot (x,y,z) = Constant. This provides methods for calculating a "distance" of a point from this plane. The distance is pseudo due to the fact that it can be negative if the point is on the non-normal side of the plane.

PlaneSide

Quaternion

Quaternion defines a single example of a more general class of hypercomplex Floats. Quaternions extends a rotation in three dimensions to a rotation in four dimensions. This aVoids "gimbal lock" and allows for smooth continuous rotation.

Ray

Ray defines a line segment which has an origin and a direction. That is, a point and an infinite ray is cast from this point. The ray is defined by the following equation: R(t) = origin + t*direction for t >= 0.

Transform

Represents a translation, rotation and scale in one object.

Triangle

Triangle defines a object for containing triangle information. The triangle is defined by a collection of three Vector3f objects.

Vector2f

Vector2f defines a Vector for a two float value vector.

Vector3f

Vector4f

Vector4f defines a Vector for a four float value tuple. Vector4f can represent any four dimensional value, such as a vertex, a normal, etc. Utility methods are also included to aid in mathematical calculations.
