class Rng

package org.angle3d.math.random

extended by Mersenne, ParkMiller, ParkMiller31

Random number generator base class.

Constructor

new ()

Variables

seed:Int

The sequence of numbers is dependent on an initial value, called the seed. This means that, if the seed isn't altered, the sequence of random numbers that is generated by the program will be exactly the same on each run of the program.

Methods

rand ():Float

Returns an integral number (see specific implementation for details).

The number is returned as a double to prevent overflow since some targets don't support unsigned 32-bit integers.

randFloat ():Float

Returns a real number in the range [0,1).

inline randFloatRange (min:Float, max:Float):Float

Returns a real number in the interval [min,max).

inline randFloatSymmetric (range:Float):Float

Returns a real number in the interval [-range,range).
