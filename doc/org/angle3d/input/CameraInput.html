class CameraInput

package org.angle3d.input

This class defines all the constants used in camera handlers for registration with the inputManager

Static variables

staticinline read onlyCHASECAM_DOWN:String

Chase camera mapping for moving down. Default assigned to MouseInput.AXIS_Y direction depending on the invertYaxis configuration

staticinline read onlyCHASECAM_MOVELEFT:String

Chase camera mapping for moving left. Default assigned to MouseInput.AXIS_X direction depending on the invertXaxis configuration

staticinline read onlyCHASECAM_MOVERIGHT:String

Chase camera mapping for moving right. Default assigned to MouseInput.AXIS_X direction depending on the invertXaxis configuration

staticinline read onlyCHASECAM_TOGGLEROTATE:String

Chase camera mapping to initiate the rotation of the cam. Default assigned to MouseInput.BUTTON_LEFT and MouseInput.BUTTON_RIGHT

staticinline read onlyCHASECAM_UP:String

Chase camera mapping for moving up. Default assigned to MouseInput.AXIS_Y direction depending on the invertYaxis configuration

staticinline read onlyCHASECAM_ZOOMIN:String

Chase camera mapping for zooming in. Default assigned to MouseInput.AXIS_WHEEL direction positive

staticinline read onlyCHASECAM_ZOOMOUT:String

Chase camera mapping for zooming out. Default assigned to MouseInput.AXIS_WHEEL direction negative
