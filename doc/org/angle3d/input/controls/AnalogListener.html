Angle3D

interface AnalogListener

package org.angle3d.input.controls

extends InputListener

implemented by ChaseCamera, FlyByCamera

AnalogListener is used to receive events of inputs in analog format.

Methods

onAnalog (name:String, value:Float, tpf:Float):Void

Called to notify the implementation that an analog event has occurred.

The results of KeyTrigger and MouseButtonTrigger events will have tpf == value.

Parameters:

name

The name of the mapping that was invoked

value

Value of the axis, from 0 to 1.

tpf

The time per frame value.
