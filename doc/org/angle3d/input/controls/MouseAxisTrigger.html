class MouseAxisTrigger

package org.angle3d.input.controls

implements Trigger

A MouseAxisTrigger is used as a mapping to mouse axis, a mouse axis is movement along the X axis (left/right), Y axis (up/down) and the mouse wheel (scroll up/down).

Constructor

new (mouseAxis:Int, negative:Bool)

Create a new MouseAxisTrigger.

Parameters:

mouseAxis

Mouse axis. See AXIS_*** constants in MouseInput

negative

True if listen to negative axis events, false if listen to positive axis events.
