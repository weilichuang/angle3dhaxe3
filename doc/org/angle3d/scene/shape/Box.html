class org.angle3d.scene.shape.Box extends AbstractBox

Available on all platforms

A box with solid (filled) faces.

Instance Fields

function new(x:Float, y:Float, z:Float, ?center:Vector3f):Void

Creates a new box.

The box has a center of 0,0,0 and extends in the out from the center by the given amount in each direction. So, for example, a box with extent of 0.5 would be the unit cube.

name

the name of the box.

x

the size of the box along the x axis, in both directions.

y

the size of the box along the y axis, in both directions.

z

the size of the box along the z axis, in both directions.
