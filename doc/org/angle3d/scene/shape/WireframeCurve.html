class org.angle3d.scene.shape.WireframeCurve extends WireframeShape

Available on all platforms

A Curve is a visual, line-based representation of a {@link Spline}. The underlying Spline will be sampled N times where N is the number of segments as specified in the constructor. Each segment will represent one line in the generated mesh.
