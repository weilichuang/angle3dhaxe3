class org.angle3d.cinematic.events.AnimationEvent extends AbstractCinematicEvent

Available on all platforms

An event based on an animation of a model. The model has to hold an AnimControl with valid animation (bone or spatial animations).

It helps to schedule the playback of an animation on a model in a Cinematic.
