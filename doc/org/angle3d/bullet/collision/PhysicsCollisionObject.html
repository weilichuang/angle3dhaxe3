function setCollisionGroup(collisionGroup:Int):Void

Sets the collision group number for this physics object.
The groups are integer bit masks and some pre-made variables are available in CollisionObject. All physics objects are by default in COLLISIONGROUP01.
Two object will collide when one of the partys has the collisionGroup of the other in its collideWithGroups set.

collisionGroup

the collisionGroup to set
