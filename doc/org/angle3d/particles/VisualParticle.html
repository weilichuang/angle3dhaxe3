class org.angle3d.particles.VisualParticle extends Particle

Available on all platforms

A VisualParticle is the most obvious implementation of a particle. It represents that particles that can be visualised on the screen.

var zRotation:Float

zRotation is used to rotate the particle in 2D (around the Z-axis) @remarks There is no relation between zRotation and orientation. rotationSpeed in combination with orientation are used for 3D rotation of the particle, while zRotation means the rotation around the Z-axis. This type of rotation is typically used for rotating textures. This also means that both types of rotation can be used together.

var zRotationSpeed:Float

The zRotationSpeed is used in combination with zRotation and defines tha actual rotationspeed in 2D.
