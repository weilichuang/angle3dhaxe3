class VisualParticle

package org.angle3d.particles

extends Particle

A VisualParticle is the most obvious implementation of a particle. It represents that particles that can be visualised on the screen.

zRotation:Float

zRotation is used to rotate the particle in 2D (around the Z-axis) @remarks There is no relation between zRotation and orientation. rotationSpeed in combination with orientation are used for 3D rotation of the particle, while zRotation means the rotation around the Z-axis. This type of rotation is typically used for rotating textures. This also means that both types of rotation can be used together.

zRotationSpeed:Float

The zRotationSpeed is used in combination with zRotation and defines tha actual rotationspeed in 2D.

_process (technique:ParticleTechnique, timeElapsed:Float):Void

Perform actions on the particle itself during the update loop of a ParticleTechnique. Active particles may want to do some processing themselves each time the ParticleTechnique is updated. One example is to perform actions by means of the registered ParticleBehaviour objects. ParticleBehaviour objects apply internal behaviour of each particle individually. They add both data and behaviour to a particle, which means that each particle can be extended with functionality.
