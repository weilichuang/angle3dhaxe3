class org.angle3d.particles.attribute.DynamicAttributeCurved extends DynamicAttribute

Available on all platforms

This is a more complex usage of the DynamicAttribute principle. This class returns a value on an curve. After setting a number of control points, this class is able to interpolate a point on the curve that is based on these control points. Interpolation is done in different flavours. 碙inear?provides linear interpolation of a value on the curve, while 碨pline?generates a smooth curve and the returns a value that lies on that curve.
