class UniformBindingManager

package org.angle3d.material.shader

UniformBindingManager helps RenderManager to manage uniform bindings.

The updateUniformBindings will update a given list of uniforms based on the current state of the manager.

setWorldMatrix (mat:Matrix4f):Void

Internal use only. Sets the world matrix to use for future rendering. This has no effect unless objects are rendered manually using Material.render. Using renderGeometry will override this value.

Parameters:

mat

The world matrix to set

updateUniformBindings (shader:Shader):Void

Internal use only. Updates the given list of uniforms with UniformBinding based on the current world state.
