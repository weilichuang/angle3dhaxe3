class org.angle3d.material.shader.UniformBindingManager

Available on all platforms

UniformBindingManager helps RenderManager to manage uniform bindings.

The updateUniformBindings will update a given list of uniforms based on the current state of the manager.

Internal use only. Sets the world matrix to use for future rendering. This has no effect unless objects are rendered manually using {@link Material#render(org.angle3d.scene.Geometry, org.angle3d.renderer.RenderManager) }. Using {@link #renderGeometry(org.angle3d.scene.Geometry) } will override this value.
