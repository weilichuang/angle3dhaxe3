Defines the basic properties of a listener associated with a Signal.

Instance Fields

var enabled:Bool

Whether the listener is called on execution. Defaults to true.

var listener:TListener

The listener associated with this slot. Note: for hxcpp 2.10 this requires a getter method to compile

var once:Bool

Whether this slot is automatically removed after it has been used once.

var priority:Int

The priority of this slot should be given in the execution order. An Signal will call higher numbers before lower ones. Defaults to 0.

function remove():Void

Removes the slot from its signal.
