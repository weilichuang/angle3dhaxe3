class SlotList<TSlot, TListener>

package msignal

Constructor

new (head:TSlot, ?tail:SlotList<TSlot, TListener>)

Creates and returns a new SlotList object.

A user never has to create a SlotList manually. Use the `NIL` element to represent an empty list. `NIL.prepend(value)` would create a list containing `value`

.

Parameters:

head

The first slot in the list.

tail

A list containing all slots except head.

Variables

head:TSlot

read onlylength:Int

The number of slots in the list.

nonEmpty:Bool

tail:SlotList<TSlot, TListener>

Methods

append (slot:TSlot):SlotList<TSlot, TListener>

Appends a slot to this list. Note: appending is O(n). Where possible, prepend which is O(1). In some cases, many list items must be cloned to avoid changing existing lists.

Parameters:

slot

The item to be appended.

Returns:

A list consisting of all elements of this list followed by slot.

contains (listener:TListener):Bool

Determines whether the supplied listener Function is contained within this list

filterNot (listener:TListener):SlotList<TSlot, TListener>

Returns the slots in this list that do not contain the supplied listener. Note: assumes the listener is not repeated within the list.

Parameters:

listener

The function to remove.

Returns:

A list consisting of all elements of this list that do not have listener.

find (listener:TListener):TSlot

Retrieves the Slot associated with a supplied listener within the SlotList.

Parameters:

listener

The Function being searched for

Returns:

The ISlot in this list associated with the listener parameter through the ISlot.listener property. Returns null if no such ISlot instance exists or the list is empty.

insertWithPriority (slot:TSlot):SlotList<TSlot, TListener>

Insert a slot into the list in a position according to its priority. The higher the priority, the closer the item will be inserted to the list head.

Parameters:

slot

The item to be inserted.

prepend (slot:TSlot):SlotList<TSlot, TListener>

Prepends a slot to this list.

Parameters:

slot

The item to be prepended.

Returns:

A list consisting of slot followed by all elements of this list.

Static variables

staticNIL:SlotList<Dynamic, Dynamic>

Represents an empty list. Used as the list terminator.
