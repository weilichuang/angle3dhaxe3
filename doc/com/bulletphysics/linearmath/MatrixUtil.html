staticdiagonalize (mat:Matrix3f, rot:Matrix3f, threshold:Float, maxSteps:Int):Void

Diagonalizes this matrix by the Jacobi method. rot stores the rotation from the coordinate system in which the matrix is diagonal to the original coordinate system, i.e., old_this = rot * new_this * rot^T. The iteration stops when all off-diagonal elements are less than the threshold multiplied by the sum of the absolute values of the diagonal, or when maxSteps have been executed. Note that this matrix is assumed to be symmetric.
