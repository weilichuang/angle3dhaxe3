class com.bulletphysics.collision.shapes.ConvexHullShape extends PolyhedralConvexShape

Available on all platforms

ConvexHullShape implements an implicit convex hull of an array of vertices. Bullet provides a general and fast collision detector for convex shapes based on GJK and EPA using localGetSupportingVertex.
