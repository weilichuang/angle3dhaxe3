class SimpleBroadphase

package com.bulletphysics.collision.broadphase

implements BroadphaseInterface

SimpleBroadphase is just a unit-test for {AxisSweep3}, {AxisSweep3_32}, or {DbvtBroadphase}, so use those classes instead. It is a brute force AABB culling broadphase based on O(n^2) AABB checks.
