class FileLoader

package assets.manager

File Load API class.

Use this class to load single or multiple files simultaneously. When files finish loading and there are no more queued, onFileLoaded signal is dispatched. Files finish loading with LOADED or ERROR status

Constructor

new (?maxConnectionLimit:Int)

Variables

manager:LoaderManager

Manager instance used to load files

onFileLoaded:Signal1<FileInfo>

Dispatched every time a file is loaded.

onFilesLoaded:Signal1<StringMap<FileInfo>>

Dispatched when files are loaded and there are no more files to load.

read onlyqueuedFiles:Array<String>

List of queued files, read only.
